# Contact Book 📒

A simple Contact Book application built using HTML, CSS, JavaScript, and Python.

This project helps users store and manage contacts by adding, viewing, searching, and deleting contact details.

## Features

✅ Add new contacts  
✅ View all saved contacts  
✅ Search contact by name  
✅ Delete contact  
✅ Interactive web interface  
✅ Python CLI version included  

---

## Technologies Used

- HTML  
- CSS  
- JavaScript  
- Python

## Web Version Features

### ➕ Add Contact
Add a new contact with:
- Name
- Phone Number

### 📜 View Contacts
Displays all saved contacts.

### 🔍 Search Contact
Search contact by name.

### 🗑️ Delete Contact
Delete contact by name.

---

## Python Version Features

CLI-based contact management system with menu options:

```text
1. Add Contact
2. View Contacts
3. Search Contact
4. Delete Contact
5. Exit
```

---

## How to Run (Web Version)

### Step 1: Clone repository

```bash
git clone <repository-link>
```

### Step 2: Open project folder

```bash
cd Contact-Book
```

### Step 3: Open HTML file

Open:

```bash
index.html
```

OR run using Live Server in Visual Studio Code.

---

## How to Run (Python Version)

Run:

```bash
python pro1.py
```

Follow menu options.
